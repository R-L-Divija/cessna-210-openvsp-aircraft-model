# Cessna 210 Aircraft Modeling – OpenVSP

This project recreates the geometry of the **Cessna 210 aircraft** using NASA’s OpenVSP software.  
The goal was to understand aircraft geometry modeling and parameterization.

## Tools Used
- OpenVSP (NASA)
- Aircraft 3-view reference drawings

## Features Modeled
- Fuselage
- Main Wing
- Horizontal Tail
- Vertical Tail
- Propeller

## Modeling Parameters
The aircraft geometry was created using common aerodynamic parameters:

- Wing span
- Chord length
- Taper ratio
- Sweep angle
- Dihedral angle
- Tail geometry
